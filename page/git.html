1. ការណែនាំអំពី Git

Git ជាប្រព័ន្ធគ្រប់គ្រងកំណែសម្រាប់តាមដានកូដ។

វាជួយអ្នកសហការនិងគ្រប់គ្រងគម្រោង។

1.1. សេចក្តីពិពណ៌នាអំពី Git

Git អនុញ្ញាតឱ្យរក្សាទុកប្រវត្តិកូដ។

ប្រើសម្រាប់គម្រោងអភិវឌ្ឍន៍កម្មវិធី។

1.2. ការដំឡើង Git

ទាញយក Git ពី git-scm.com។

ដំឡើងនៅលើ Windows, Mac, ឬ Linux។

# ពិនិត្យកំណែ Git
git --version

សាកល្បង៖

ពិនិត្យកំណែ Git។

1.3. ការកំណត់ Git

កំណត់ឈ្មោះនិងអ៊ីមែលរបស់អ្នក។

ប្រើសម្រាប់កំណត់អត្តសញ្ញាណ commit។

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

2. ការចាប់ផ្តើម Repository

បង្កើត repository ដើម្បីតាមដានកូដ។

ប្រើ git init នៅថតគម្រោង។

git init

សាកល្បង៖

ចាប់ផ្តើម repository ថ្មី។

3. ការបញ្ជូន Commit

Commit រក្សាទុកការផ្លាស់ប្តូរកូដ។

ប្រើ git add និង git commit។

git add .
git commit -m "Initial commit"

4. សាខា (Branch)

សាខាអនុញ្ញាតឱ្យធ្វើការលើមុខងារថ្មី។

បង្កើតសាខាដោយ git branch។

git branch feature
git checkout feature

សាកល្បង៖

បង្កើតនិងប្តូរទៅសាខា feature។

5. ការបញ្ចូលសាខា (Merge)

Merge បញ្ចូលការផ្លាស់ប្តូរពីសាខាមួយទៅសាខាមួយទៀត។

ប្រើ git merge។

git checkout main
git merge feature

6. Remote Repository

Remote repository រក្សាទុកកូដនៅលើ GitHub។

ភ្ជាប់ជាមួយ git remote add។

git remote add origin https://github.com/user/repo.git

7. Push និង Pull

Push បញ្ជូនកូដទៅ remote; pull ទាញកូដមក។

ប្រើ git push និង git pull។

git push origin main
git pull origin main

8. ការដោះស្រាយបញ្ហាប៉ះទង្គិច

បញ្ហាប៉ះទង្គិចកើតឡើងនៅពេល merge។

កែសម្រួលឯកសារដោយដៃ ឬប្រើឧបករណ៍។

git merge feature
# កែឯកសារប៉ះទង្គិច
git add .
git commit

សាកល្បង៖

Merge សាខា feature ទៅ main។

9. Stash ការផ្លាស់ប្តូរ

Stash រក្សាទុកការផ្លាស់ប្តូរបណ្តោះអាសន្ន។

ប្រើ git stash និង git stash pop។

git stash
git stash pop

10. ការអនុវត្តល្អបំផុត

សរសេរសារ� commit ច្បាស់លាស់។

ប្រើសាខាសម្រាប់មុខងារនីមួយៗ។

git commit -m "Add user authentication"
