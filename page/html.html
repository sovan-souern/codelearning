1. ការណែនាំអំពី HTML

HTML ជាភាសាសម្រាប់បង្កើតគេហទំព័រនៅលើអ៊ីនធឺណិត។

វាកំណត់រចនាសម្ព័ន្ធនិងខ្លឹមសារនៃទំព័របណ្តាញ។

1.1. សេចក្តីពិពណ៌នាអំពី HTML

HTML ជា HyperText Markup Language សម្រាប់រៀបចំទំព័រ។

ប្រើស្លាកដើម្បីកំណត់ធាតុដូចជា អត្ថបទ និងរូបភាព។

1.2. ប្រវត្តិសង្ខេប HTML

HTML បង្កើតដោយ Tim Berners-Lee ក្នុងឆ្នាំ 1990។

HTML5 ជាកំណែទំនើបចុងក្រោយបំផុត។

1.3. រចនាសម្ព័ន្ធ HTML

ឯកសារ HTML ចាប់ផ្តើមដោយ <!DOCTYPE html>។

មានស្លាក <html>, <head>, <body>។

<!DOCTYPE html>
<html>
<head>
    <title>ទំព័រដំបូង</title>
</head>
<body>
    <h1>សួស្តី!</h1>
</body>
</html>

សាកល្បង៖

បង្កើតទំព័របង្ហាញ "សួស្តី ខ្ញុំរៀន HTML!"

1.4. កម្មវិធីនិពន្ធ HTML

ប្រើ VS Code, Notepad++ ឬ Sublime Text។

រក្សាទុកឯកសារជា .html។

2. ស្លាក HTML

ស្លាកកំណត់ធាតុនៅក្នុងទំព័របណ្តាញ។

ឧទាហរណ៍៖ <p>, <h1>, <div>។

2.1. ស្លាកជាមូលដ្ឋាន

ស្លាក <h1> ដល់ <h6> សម្រាប់ចំណងជើង។

<p> សម្រាប់កថាខណ្ឌអត្ថបទ។

<h1>ចំណងជើង</h1>
<p>អត្ថបទ</p>

2.2. គុណលក្ខណៈ

គុណលក្ខណៈផ្តល់ព័ត៌មានបន្ថែមដល់ស្លាក។

ឧទាហរណ៍៖ id, class, style។

<p id="text">អត្ថបទ</p>

2.3. ការធ្វើទ្រង់ទ្រាយអត្ថបទ

ប្រើ <b>, <i> សម្រាប់ដិតនិងទ្រេត។

<strong>, <em> សម្រាប់សង្កត់ធ្ងន់។

<b>ដិត</b> <i>ទ្រេត</i>

សាកល្បង៖

បង្កើតចំណងជើងនិងអត្ថបទដិត។

3. តំណភ្ជាប់

ស្លាក <a> បង្កើតតំណភ្ជាប់ទៅទំព័រផ្សេង។

ប្រើគុណលក្ខណៈ href សម្រាប់អាសយដ្ឋាន។

<a href="https://example.com">ចុចទីនេះ</a>

4. រូបភាព

ស្លាក <img> បង្ហាញរូបភាព។

ប្រើ src និង alt សម្រាប់បរិយាយ។

<img src="image.jpg" alt="រូបភាព">

5. បញ្ជី

បញ្ជីមាន <ul> (គ្មានលំដាប់) និង <ol> (មានលំដាប់)។

ធាតុបញ្ជីប្រើ <li>។

<ul>
    <li>ធាតុ 1</li>
    <li>ធាតុ 2</li>
</ul>

6. តារាង

ស្លាក <table> បង្កើតតារាងទិន្នន័យ។

ប្រើ <tr>, <th>, <td>។

<table>
    <tr>
        <th>ក្បាល</th>
        <td>ទិន្នន័យ</td>
    </tr>
</table>

7. ទម្រង់បញ្ចូល

ស្លាក <form> បង្កើតទម្រង់សម្រាប់ទិន្នន័យ។

ប្រើ <input>, <label>។

<form>
    <label>ឈ្មោះ:</label>
    <input type="text">
</form>

សាកល្បង៖

បង្កើតទម្រង់បញ្ចូលឈ្មោះ។

8. ស្លាកសម័យទំនើប

ស្លាក HTML5 ដូចជា <header>, <footer>។

ធ្វើឱ្យកូដច្បាស់លាស់និងងាយស្វែងរក។

<header>បឋមកថា</header>
<footer>បាតកថា</footer>

9. ភាពអាចចូលប្រើបាន

ប្រើ alt, aria-label សម្រាប់អ្នកប្រើពិការ។

ធ្វើឱ្យគេហទំព័រងាយប្រើសម្រាប់មនុស្សគ្រប់គ្នា។
