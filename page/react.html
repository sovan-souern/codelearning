1. ការណែនាំអំពី React

React ជា JavaScript library សម្រាប់បង្កើត user interfaces ដែលមានភាពអន្តរកម្ម។

វាត្រូវបានបង្កើតដោយ Facebook និងប្រើសម្រាប់គម្រោងទំនើប។

1.1. សេចក្តីពិពណ៌នាអំពី React

React ប្រើ component-based architecture ដើម្បីបង្កើត UI ដែលអាចប្រើឡើងវិញបាន។

វាមាន virtual DOM ដែលធ្វើឱ្យការបង្ហាញទិន្នន័យមានល្បឿនលឿន។

1.2. ការដំឡើង React
