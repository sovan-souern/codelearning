1. ការណែនាំអំពី PHP

PHP ជាភាសាសរសេរកម្មវិធីខាងម៉ាស៊ីនមេសម្រាប់បង្កើតគេហទំព័រថាមវន្ត។

វាត្រូវបានប្រើប្រាស់យ៉ាងទូលំទូលាយនៅក្នុងប្រព័ន្ធគ្រប់គ្រងខ្លឹមសារដូចជា WordPress។

1.1. សេចក្តីពិពណ៌នាអំពី PHP

PHP មានន័យថា "Hypertext Preprocessor" និងជាភាសាបើកចំហ។

វាអនុញ្ញាតឱ្យបង្កប់កូដ PHP នៅក្នុង HTML ដើម្បីបង្កើតខ្លឹមសារថាមវន្ត។

1.2. ប្រវត្តិសង្ខេប PHP

PHP ត្រូវបានបង្កើតឡើងដោយ Rasmus Lerdorf ក្នុងឆ្នាំ 1994។

PHP 8 ជាកំណែទំនើបចុងក្រោយបំផុត (គិតត្រឹមឆ្នាំ 2025)។

1.3. ការរៀបចំបរិស្ថាន

ដំឡើង XAMPP ឬ WAMP ដើម្បីបង្កើតម៉ាស៊ីនមេ PHP នៅលើកុំព្យូទ័ររបស់អ្នក។

រក្សាទុកឯកសារ PHP ជាមួយនឹងផ្នែកបន្ថែម .php នៅក្នុងថត htdocs។

សាកល្បង៖

បង្កើតឯកសារ hello.php ដែលបង្ហាញ "សួស្តី ខ្ញុំរៀន PHP!" នៅក្នុង XAMPP ហើយចូលទៅកាន់ http://localhost/hello.php។

2. អថេរនិងប្រភេទទិន្នន័យ

អថេរនៅក្នុង PHP ចាប់ផ្តើមដោយ $ ហើយមិនតម្រូវឱ្យប្រកាសប្រភេទ។

ប្រភេទទិន្នន័យរួមមាន String, Integer, Float, Boolean, Array, Object។

3. ប្រតិបត្តិករ

PHP មានប្រតិបត្តិករគណនា (+, -, *, /), ប្រៀបធៀប (==, ===, !=), និងតក្កវិជ្ជា (&&, ||)។

សាកល្បង៖

បង្កើតឯកសារ sum.php ដែលគណនាផលបូកនៃ 10 និង 5 ហើយបង្ហាញលទ្ធផល។

4. រចនាសម្ព័ន្ធគ្រប់គ្រង

ប្រើ if, else, elseif, switch សម្រាប់លក្ខខណ្ឌ។

រង្វិលជុំរួមមាន for, while, foreach។

5. អនុគមន៍
