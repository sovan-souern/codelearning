1. ការណែនាំអំពី Tailwind CSS

Tailwind CSS ជា CSS framework ដែលប្រើ utility-first approach ដើម្បីរចនាគេហទំព័រយ៉ាងរហ័ស។

វាផ្តល់នូវ classes ដែលអាចប្រើភ្លាមៗដើម្បីកំណត់រចនាបទ។

1.1. សេចក្តីពិពណ៌នាអំពី Tailwind CSS

Tailwind CSS មិនប្រើ predefined components ដូច Bootstrap ទេ ប៉ុន្តែផ្តល់ភាពបត់បែនខ្ពស់។

វាស័ក្តិសមសម្រាប់គម្រោងដែលត្រូវការរចនាបទផ្ទាល់ខ្លួន។

1.2. ការដំឡើង Tailwind CSS

ប្រើ CDN ឬដំឡើងតាមរយៈ npm ដើម្បីប្រើ Tailwind CSS។

ឧទាហរណ៍ បន្ថែម CDN ទៅក្នុង HTML។

<link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">

សាកល្បង៖

បន្ថែម Tailwind CSS CDN ទៅក្នុង HTML។

2. ការប្រើ Utility Classes

Utility classes អនុញ្ញាតឱ្យអ្នកកំណត់រចនាបទដោយផ្ទាល់នៅក្នុង HTML។

ឧទាហរណ៍ កំណត់ padding និង margin។

<div class="p-4 m-2 bg-blue-500 text-white">សួស្តី Tailwind!</div>

សាកល្បង៖

បង្កើត div ដែលមាន padding, margin និងពណ៌ផ្ទៃខាងក្រោយ។

3. ប្លង់និងការរៀបចំ

Tailwind ផ្តល់ classes សម្រាប់ flexbox, grid, និង spacing។

ឧទាហរណ៍ បង្កើត flex container។

<div class="flex space-x-4">
    <div class="bg-red-500 p-4">ធាតុ 1</div>
    <div class="bg-green-500 p-4">ធាតុ 2</div>
</div>

4. អក្សរសាស្ត្រ

កំណត់ទំហំអក្សរ ពណ៌ និងទម្ងន់ដោយប្រើ classes ដូចជា text- និង font-។

ឧទាហរណ៍ កំណត់ heading។

<h1 class="text-3xl font-bold text-gray-800">ចំណងជើង</h1>

សាកល្បង៖

កំណត់រចនាបទអក្សរសម្រាប់ heading។

5. ពណ៌និងផ្ទៃខាងក្រោយ

ប្រើ classes ដូចជា bg- និង text- ដើម្បីកំណត់ពណ៌។

ឧទាហរណ៍ ផ្លាស់ប្តូរពណ៌ផ្ទៃខាងក្រោយ។

<div class="bg-gradient-to-r from-blue-500 to-purple-500 p-6 text-white">Gradient Background</div>

6. ការឆ្លើយតប (Responsive Design)

ប្រើ prefixes ដូចជា sm:, md:, lg: សម្រាប់ responsive design។

ឧទាហរណ៍ ផ្លាស់ប្តូរទំហំអក្សរតាមទំហំអេក្រង់។

<div class="text-base md:text-lg lg:text-xl">អក្សរដែលឆ្លើយតប</div>

7. សមាសធាតុនិងការបង្កើតផ្ទាល់ខ្លួន

បង្កើតសមាសធាតុដោយប្រើ utility classes។

ឧទាហរណ៍ ប៊ូតុង។

<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">ចុចខ្ញុំ</button>

សាកល្បង៖

បង្កើតប៊ូតុងដែលមាន hover effect។

8. ការប្រើ Directives និង Functions

ប្រើ @apply និង functions ដើម្បីបង្កើត custom styles។

ឧទាហរណ៍ កំណត់ custom class។

/* tailwind.config.js */
@tailwind base;
@tailwind components;
.btn-custom {
    @apply bg-blue-500 text-white font-bold py-2 px-4 rounded;
}
@tailwind utilities;

9. ការប្តូរតាមបំណង

កែប្រែ Tailwind configuration ដើម្បីបន្ថែមពណ៌ ទំហំ ឬ themes។

ឧទាហរណ៍ បន្ថែមពណ៌ផ្ទាល់ខ្លួន។

module.exports = {
    theme: {
        extend: {
            colors: {
                'custom-blue': '#1e40af',
            }
        }
    }
}

10. ការអនុវត្តល្អបំផុត

ប្រើ utility classes ដើម្បីរក្សាកូដឱ្យសាមញ្ញ និងអាចថែទាំបាន។

កំណត់ custom components និងប្រើ PurgeCSS ដើម្បីកាត់បន្ថយទំហំឯកសារ CSS�।

<div class="container mx-auto p-4">រក្សាកូដឱ្យស្អាត</div>
