ផ្ទាំង C++

កម្មវិធី C++ គឺជាភាសាសរសេរកម្មវិធីដែលមានឥទ្ធិពល និងត្រូវបានគេប្រើយ៉ាងទូលំទូលាយក្នុងការអភិវឌ្ឍន៍កម្មវិធីកុំព្យូទ័រ។

ចូលមើលបន្ថែម
ថ្នាក់ HTML

កម្មវិធី HTML គឺជាឧបករណ៍ដែលត្រូវបានប្រើប្រាស់សម្រាប់សរសេរកូដ HTML ដើម្បីបង្កើតគេហទំព័រ និងការបង្ហាញព័ត៌មាននៅលើបណ្ដាញ។

ចូលមើលបន្ថែម
ថ្នាក់ CSS

CSS (Cascading Style Sheets) គឺជាភាសាកំណត់រចនាសម្ព័ន្ធសម្រាប់ការរចនាឬការតុបតែងគេហទំព័រជាមួយ HTML។

ចូលមើលបន្ថែម
ថ្នាក់ JavaScript

កម្មវិធី JavaScript គឺជាភាសាសរសេរកម្មវិធីដែលត្រូវបានគេប្រើយ៉ាងទូលំទូលាយសម្រាប់ការអភិវឌ្ឍន៍គេហទំព័រ និងកម្មវិធីផ្សេងៗទៀត។

ចូលមើលបន្ថែម
ថ្នាក់ Python

កម្មវិធី Python គឺជាភាសាកម្មវិធីកម្រិតខ្ពស់ដែលមានលក្ខណៈសាមញ្ញ អានយល់បានងាយ និងមានកម្លាំងខ្លាំងក្នុងការអភិវឌ្ឍកម្មវិធី។

ចូលមើលបន្ថែម
ថ្នាក់ Git

កម្មវិធី Git គឺជាកម្មវិធី Version Control System (VCS) ដែលត្រូវបានប្រើដើម្បីគ្រប់គ្រងកូដប្រភព និងតាមដានការផ្លាស់ប្តូររបស់វា។

ចូលមើលបន្ថែម
ថ្នាក់ MySQL

MySQL គឺជាប្រព័ន្ធគ្រប់គ្រងមូលដ្ឋានទិន្នន័យប្រភេទបំណែកទំនាក់ទំនង (RDBMS) ដែលគ្រប់គ្រងការផ្ទុក និងទាញយកទិន្នន័យ។

ចូលមើលបន្ថែម
ថ្នាក់ React

React គឺជា បណ្ណាល័យ JavaScript (JavaScript Library) ដែលប្រើសម្រាប់បង្កើត ចំណុចប្រទាក់អ្នកប្រើប្រាស់ (UI)។

ចូលមើលបន្ថែម
ថ្នាក់ PHP

PHP គឺជាភាសាសរសេរកម្មវិធីដែលត្រូវបានគេប្រើយ៉ាងទូលំទូលាយសម្រាប់ការអភិវឌ្ឍន៍គេហទំព័រ។
