1. ការណែនាំអំពី CSS

CSS ជាភាសាសម្រាប់រចនារូបរាងគេហទំព័រ។

វាគ្រប់គ្រងពណ៌ ទំហំ និងប្លង់។

1.1. សេចក្តីពិពណ៌នាអំពី CSS

CSS ឈរជើងជា Cascading Style Sheets។

ប្រើសម្រាប់ធ្វើឱ្យ HTML មានភាពស្រស់ស្អាត។

1.2. វាក្យសម្ព័ន្ធ CSS

CSS មានកម្រិតជ្រើសរើស និងលក្ខណៈសម្បត្តិ។

ឧទាហរណ៍៖ selector { property: value; }

h1 {
    color: blue;
}

សាកល្បង៖

ធ្វើឱ្យ h1 មានពណ៌ក្រហម។

1.3. ការបញ្ចូល CSS

បញ្ចូល CSS តាមរយៈ <link>, <style>, ឬ inline។

ឧទាហរណ៍៖ <link rel="stylesheet" href="style.css">

<link rel="stylesheet" href="style.css">

2. កម្រិតជ្រើសរើស

កម្រិតជ្រើសរើសជ្រើសធាតុ HTML ដើម្បីរចនា។

ឧទាហរណ៍៖ ឈ្មោះស្លាក, id, class។

.text {
    font-size: 16px;
}

3. គំរូប្រអប់

គំរូប្រអប់មាន margin, padding, border។

គ្រប់គ្រងទំហំនិងចន្លោះធាតុ។

div {
    margin: 10px;
    padding: 15px;
    border: 1px solid black;
}

សាកល្បង៖

បន្ថែម padding និង border ទៅ div។

4. ពណ៌និងផ្ទៃខាងក្រោយ

ប្រើ color និង background សម្រាប់រចនា។

ឧទាហរណ៍៖ ពណ៌អក្សរ ឬរូបភាពផ្ទៃខាងក្រោយ។

p {
    color: #333;
    background-color: #f0f0f0;
}

5. អក្សរសាស្ត្រ

គ្រប់គ្រងអក្សរជាមួយ font-family, font-size។

ឧទាហរណ៍៖ កំណត់រចនាប័ទ្មអក្សរ។

h2 {
    font-family: 'Arial', sans-serif;
    font-size: 24px;
}

6. Flexbox

Flexbox រៀបចំប្លង់ដោយបត់បែន។

ប្រើ display: flex និង justify-content។

.container {
    display: flex;
    justify-content: space-between;
}

សាកល្បង៖

រៀបចំធាតុជាមួយ Flexbox។

7. Grid

CSS Grid បង្កើតប្លង់ជាក្រឡា។

ប្រើ display: grid និង grid-template-columns។

.grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
}

8. ការរចនាឆ្លើយតប

ប្រើ media queries សម្រាប់អេក្រង់ផ្សេងៗ។

ឧទាហរណ៍៖ ផ្លាស់ប្តូរទំហំអក្សរនៅទូរស័ព្ទ។

@media (max-width: 600px) {
    h1 {
        font-size: 20px;
    }
}

9. ចលនា

ប្រើ transition និង @keyframes សម្រាប់ចលនា។

ឧទាហរណ៍៖ ផ្លាស់ប្តូរពណ៌នៅពេលចុច។

button:hover {
    background-color: blue;
    transition: background-color 0.3s;
}

សាកល្បង៖

បន្ថែមចលនា transition ទៅប៊ូតុង។

10. ភាពអាចចូលប្រើបាន

ប្រើ :focus និង outline សម្រាប់អ្នកប្រើពិការ។

ធានាពណ៌មានកម្រិតផ្ទុយគ្នាខ្ពស់។

a:focus {
    outline: 2px solid blue;
}

11. CSS Variables

CSS Variables (Custom Properties) អាចប្រើសម្រាប់រក្សាតម្លៃដែលអាចប្រើឡើងវិញ។

ប្រើ --variable-name ដើម្បីប្រកាស និង var(--variable-name) ដើម្បីប្រើ។

:root {
    --main-color: #3498db;
    --padding: 10px;
}
button {
    background-color: var(--main-color);
    padding: var(--padding);
}

សាកល្បង៖

ប្រើ CSS Variables ដើម្បីកំណត់ពណ៌ និង Padding សម្រាប់ប៊ូតុង។

12. CSS Transforms

CSS Transforms អាចប្រើសម្រាប់ផ្លាស់ប្តូររូបរាងធាតុ។

ប្រើ transform ដើម្បីបង្វិល, បង្រួម, ឬផ្លាស់ទីធាតុ។

div {
    transform: rotate(45deg) scale(1.2);
}

សាកល្បង៖

បង្វិល និងបង្រួមធាតុ div ដោយប្រើ transform។
